User Info Microservice

About:

A practice project from the curriculum of freeCodeCamp. A Node/Express project that returns the IP address, language, and operating system of the requester. The source code for the entire project can be seen on GitHub here: https://github.com/hiltydiggs/user-info-microservice

Usage:

URL: http://user-info-microservice.azurewebsites.net/get-info/

Returns:
{
  "ip-address": "50.186.XXX.XX:XXXXX",
  "language": "en-US",
  "operating-system": "Windows 10 0.0.0",
  "browser": "Chrome 52.0.2743"
}
